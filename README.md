## 📖 Overview

The **Personal Expense Tracker** is a Python-based utility designed for users who want a simple way to track daily expenditures without bloated software. It logs your data into a CSV format and generates high-quality visual reports to help you understand where your money goes.

## 🚀 Key Features

* ✅ **Instant Logging**: Record expenses with categories, amounts, and descriptions in seconds.
* 📈 **Visual Analytics**: Automatically generates a professional pie chart using `Matplotlib`.
* 🗂️ **CSV Storage**: Data is stored in a standard CSV format, making it easy to export to Excel or Google Sheets.
* 🤖 **Simple CLI**: Clean, menu-driven interface for a seamless user experience.
